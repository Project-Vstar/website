## About

This repository contains the source code for the VSTAR company website. The site serves as the central hub for talent profiles, lore, audition information, press resources, and official announcements.

## What We Do

- **Live Streaming** — 2D & 3D gaming, ASMR, singing, chatting, and collab streams on YouTube, Twitch, X, and TikTok. Full-scale 3D live events and online concerts.
- **Entertainment Technology** — Partnering with tech and communications companies to bring cutting-edge experiences to anime and gaming conventions worldwide.
- **Other Media** — Original manga and stories set in the VINFERNIA & VSTAR universe. Future projects include a video game and an animated series.

## Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) with Turbopack
- **UI:** React 19, [Tailwind CSS v4](https://tailwindcss.com/)
- **Carousel:** Edited [Embla Carousel](https://www.embla-carousel.com/)
- **Analytics:** [Vercel Analytics](https://vercel.com/analytics)
